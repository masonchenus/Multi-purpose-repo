# Game Center Version 2.0.2

**Release Date:** December 24, 2025

## Update Summary

Version 2.0.2 introduces permanent score storage with database integration. This release marks the transition from in-memory storage to persistent data management.

## New Features

- PostgreSQL database integration
- Persistent score storage across server restarts
- User account system with authentication
- Score history tracking

## Technical Changes

- Migrated from in-memory arrays to PostgreSQL
- Implemented JWT authentication
- Added database schema management
- Improved API response times with indexing

## Improvements

- Data recovery from unexpected shutdowns
- Better user session management
- Enhanced security with password hashing
